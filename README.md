# RhythmFlow - Finished App (Playable + Playlists)

This Expo project contains:
- Library with 5 playable sample tracks (remote MP3s)
- Player screen with play/pause/prev/next controls (expo-av)
- Playlists screen: create playlists, add sample tracks, play from playlist
- Centralized GlobalStyles.js used by all screens
- AsyncStorage used to persist playlists

How to run:
1. npm install
2. expo start
